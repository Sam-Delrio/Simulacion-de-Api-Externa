Actividad Experiencial 2
se conecta un frontend en React con una API REST local usando JSON Server. Permite crear, editar, eliminar y marcar tareas como completadas

# **Cómo ejecutar el proyecto**
Iniciar la API (Backend)

## **Entrar a la carpeta del backend:**
```
cd backend
```

## **Ejecutar el servidor JSON:**
```
npx json-server --watch db.json --port 4000
```

## **La API queda activa en:**
```
http://localhost:4000/tasks
```
## **Iniciar el Frontend**

## **En otra terminal, entrar a la carpeta del frontend:**
```
cd frontend
```

## **Instalar dependencias:**
```
npm install
```

## **Correr la aplicación:**
```
npm run dev
```

Se abre en:
```
http://localhost:5173/
```
# **Funcionalidades**

-Crear tareas

-Editar tareas

-Eliminar tareas

-Marcar como completadas

-API REST funcionando con JSON Server

-React Router + Context APIa
